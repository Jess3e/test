Librairie Arbre Binaire
==========================================

Cette librairie implémente un arbre binaire qui contient une valeur et qui peut 
avoir aucun enfant, un enfant ou deux enfants. Ces enfants peuvent à leur tour
avoir leurs enfants et une valeur.

Compiler la librairie
---------------------

  - Tout compiler:
		
***

	make

***

  - Pour les autres types de compilation (Debuggage seulement, statique
		seulement, etc.), voir le fichier Makefile.


Éditeur
-------

  - Utiliser un éditeur de texte pour modifier les fichiers.
